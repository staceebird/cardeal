QT Hairstyles

QT Hairstyles

Class: WEB 2500

Professor: Noel Alton

Technologies: Adobe Xd, Visual Studio Code, HTML5, CSS, SASS, Unsplash

Take a Look: https://quizzical-chandrasekhar-6c862e.netlify.app/

For the colors I stuck with coral #EA9E7F, light coral #F6C4AF, black #000, light-gray #C4C0BD, and gray #746E69. The font I used was Baskerville Old Face.

This was my final project for WEB 2500. Throughout the semester we created this project, we had to get users to complete tasks and fill out forms to learn how a user will use a website. I had to create a persona for the audience my webpage targets. By doing this, it helped me know what was most important to include. It is more on the simple side when it comes to websites but it does the job of teaching a user how to do a simple hairstyle.

The cards and navbar were not lining up correctly but then I learned that I needed to change the width of my logo to “rem” which fixed my navbar issue. Then for the cards I needed to retype the code because I messed around with them too much that I ended up breaking the code. Once I changed the code, my cards were more balanced.

This website taught me how to make it more user friendly and to listen to the users more because I am not making the website for my use but for others to use. When you get feedback from others, you are able to improve your skills much more.
