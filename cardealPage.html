Cardeal

Cardeal

Class: WEB 2890

Professor: Laura MacLeod

Technologies: Visual Studio Code, Bootstrap 5.0, HTML5, CSS3, SASS, Javascript

Take a Look: https://icarus.cs.weber.edu/~sb91926/bird_2890proj/

For the colors I stuck with blue #0000ff, dark blue rgb(0, 0, 90), black #000, light-gray #e9ecef, and gray #6c757d. The fonts I used were Poppins and Raleway.

This was my first project in WEB 2890 that I did on my own. The website is called Cardeal. It is a fictitious company that will buy your car and sell it online. I was given the logo but had to color it myself using https://www.vecteezy.com/editor and then the other images I found on Unsplash. I have my header/nav, hero section, how it works, advantages of selling your car to Cardeal, testimonials section, recent purchases, short section which features the main selling point, FAQ section, and the footer.



For the how it works section, I used numbers and arrows to indicate the steps. For the testimonials section I used a carousel. For this the height would adjust and I had to create a fixed height to keep it from changing heights. I used an accordion feature for the FAQ section. I went to Free Fontend to find ideas on how to create a nice looking accordion.

For the short section I created a form that would popup so the user could have another chance to send their car to Cardeal. I was struggling with the height and centering the form but once I changed the height and the position needed to be changed to “fixed”. After I sent the project to my professor I received feedback about how the image in the hero section should have the overlay removed to show the original color. I also needed to change the height with BY USING FLEX.

I was proud of the popup form and even how the hero section turned out. I have never created something like in my other web classes and I feel like he made it look more professional. I did also like that with the recent purchases section, there is a button for the user to click to show more information instead of showing everything all at once.
