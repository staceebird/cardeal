Rock Paper Scissors - Vue

Javascript Rock Paper Scissors Using Vue

Class: WEB 2630

Professor: Cody Squadroni

Technologies: Visual Studio Code, VueJS GitHub, Netlify

Take a Look: https://distracted-hawking-b46fd5.netlify.app/

Project Description: For this project I created a classic game of Rock, Paper, Scissors using VueJS. I have included these in this project.

  • el / data
  • methods / computed
  • v-bind (:) / v-on (@click)
  • :class / :style
  • v-if / v-else / v-for / v-show

    • The user is able to pick which one they want to use (Rock, Paper, or Scissors) then the computer choses one and the results are outputted. The progress bar increases either on the User or Computer side to show the wins. Once it hits 10, it resets itself.
